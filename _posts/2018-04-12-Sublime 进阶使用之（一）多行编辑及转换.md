一段文字如下
```text
Mon
Tue
Wed
```

转换成
```text
["Mon", "Tue", Wed"]
```

转换步骤

1， 打开`sublime`， 输入如下内容
```
Mon
Tue
Wed
```

2, 全选如上内容 `Ctrl` + `A`, 选中后按 `CTRL` + `SHIFT` + `L`

3, 在行尾 `",` ，在行头加上`"`

4, 第一行加上 `[`, 删除最后一个 `,`, 加上 `]`

5, 选中所有内容， `Ctrl` + `J`, 合并成一行

```text
["Mon", "Tue", Wed"]
```

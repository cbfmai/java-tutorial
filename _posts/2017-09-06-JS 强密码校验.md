强密码

```js
var strongRegex = new RegExp("^(?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])(?=.*[!@#\$%\^&\*])(?=.{8,})");
```

| 表达式        | 描述   |  
| --------   | :-----|
| ^     | 字符串开始 |
| (?=.*[a-z])        |   必须包含至少一个小写字母   |  
| (?=.*[A-Z])        |   必须包含至少一个大写字母     |
| (?=.*[0-9])    | 必须包含至少一个0-9中的一个数字 |
| (?=.*[!@#\$%\^&\*])        |  必须包含至少列举中的一个特殊字符  |  
|(?=.{8,})    |    至少8位   |
